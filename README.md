## Engine for the emoji whisperer npm package

Constructing the engine that will find the perfect emoji using publically available examples of emoji use.

Constructs inverted index and a small-scale search engine to preprocess and rank emojis. 

Exploring possibilities of sorting these emojis and how they should integrate with the user experience
